\begin{center}
  \textbf{\texttt{Git}} is a version control system (VCS) for tracking changes in
  computer files and coordinating work on those files among multiple
  people and \textbf{\texttt{\texttt{Github}}} is a free (of cost) web-based Git repository and
  internet hosting service gaining popularity among FOSS developers.
  
\emph{The present discussion is an attempt to provide a tutorial
  introduction to using git locally for version controlling
  projects. Additionally, the session will provide an introduction to
  contributing to/setting up collaborative projects on github.} 
\end{center}
The session will devote the first 30 minutes introducing \texttt{git}
and the last 30 minutes in setting up and contributing to a
collaborative project on \texttt{github}.
